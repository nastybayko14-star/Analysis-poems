# Analyzer Project of the text corpus of poems Okudzhava

## Описание
Программа для лингвистического анализа текстового корпуса стихотворений Б.Ш. Окуджавы.

## Функции
- **Анализ текста**: подсчёт слов, уникальных слов, расчёт коэффициента TTR[citation:10].
- **Работа с файлами**: чтение `.txt` файлов, сохранение отчётов.
- **Генерация отчётов**: создание сводки в форматах `.txt` и `.csv`.

## Установка и запуск
1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/nastybayko14-star/Analysis-poems.git
    ```
2.  Перейдите в папку проекта и запустите:
    ```bash
    cd text_corpus_analyser
    python main.py
    ```

## Структура проекта
- `main.py` — главный модуль
- `text_utils.py` — функции анализа текста
- `file_utils.py` — функции для работы с файлами
